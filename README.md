# 🚩 Cyber Security Writeups & CTF Archive

ยินดีต้อนรับสู่คลังรวบรวม **Writeups** และผลการวิเคราะห์จากการทดสอบเจาะระบบ (Penetration Testing) รวมถึงการแข่งขัน **CTF (Capture The Flag)** ในรายการต่าง ๆ ที่ผมได้เข้าร่วม

---

### 📂 Repository Structure

ภายใน Repository นี้ประกอบไปด้วย Writeups จากรายการสำคัญ ดังนี้:

- **🏆 Cyber Apocalypse (HackTheBox)**: รวบรวมทางแก้โจทย์จากปี 2022 - 2025
- **🛡️ CTF Archive (2018 - Present)**: คลังเก็บ Writeups เก่าแก่ตั้งแต่ปี 2018 จนถึงปัจจุบัน แบ่งตามปีและรายการแข่งขัน
- **💻 CyberWars TTX**: ข้อมูลและการวิเคราะห์จากการฝึกซ้อม Tabletop Exercise
- **🌐 Google CTF**: แนวทางการทำโจทย์จากรายการ Google CTF
- **🚀 NahamCon CTF**: Writeups จากรายการ NahamCon (2023, 2025)
- **👻 Hack The Boo**: โจทย์ธีม Halloween จาก HTB

---

### 🛠️ Key Focus Areas

การวิเคราะห์ใน Writeups เหล่านี้เน้นทักษะในด้าน:
- **Digital Forensics & Incident Response (DFIR)**
- **Reverse Engineering & Malware Analysis**
- **Web Exploitation & Network Security**
- **Cryptography & Steganography**
