### ID 106: Hosting Gambling Website - 05
**Question:** หลังจากที่ผู้ไม่หวังดีดำเนินเปลี่ยนแปลงหน้าเว็บไซต์สำเร็จแล้ว อยากทราบว่า User-Agent ที่ผู้ไม่หวังดีใช้ในการทดสอบเข้ามาเรียกใช้งานหน้าเว็บไซต์ที่ทำการเปลี่ยนแปลงแล้วนั้นคืออะไร
**Answer:** `Mozilla/5.0 (X11; Linux x86_64; rv:128.0) Gecko/20100101 Firefox/128.0`
**Investigation:**
จากการตรวจสอบ Access Log หลังจากที่มีการใช้คำสั่ง `sed` แก้ไขไฟล์ `index.php` พบว่ามีการเรียกใช้งานไฟล์ `gambling.php` และ `gam.jpg` จาก IP `199.127.63.155` โดยระบุ User-Agent เป็นเบราว์เซอร์ Firefox เวอร์ชัน 128 บน Linux

### ID 108: Hosting Gambling Website - 06
**Question:** ผู้ไม่หวังดีได้มีการรันคำสั่งเพื่อแก้ไขไฟล์หน้าเว็บหลัก และเนื่องจากการโจมตีครั้งนี้ไม่ได้มาจากผู้ไม่หวังดีคนเดิมซึ่งเป็น IP Address ใหม่ ให้ระบุ IP Address ของผู้ไม่หวังดีหลักในเหตุการณ์นี้
**Answer:** `199.127.63.155`
**Investigation:**
จากการตรวจสอบ Access Log ในช่วงเวลาที่มีการแก้ไขไฟล์หน้าหลัก (`index.php`) และดาวน์โหลดไฟล์มัลแวร์มาติดตั้ง พบว่าการโจมตีเปลี่ยนมาจาก IP `103.124.60.4` เป็น IP `199.127.63.155` ซึ่งเป็น IP หลักที่ดำเนินกระบวนการ Defacement จนสำเร็จ

### ID 110: Hosting Gambling Website - 07

**Question:** ผู้ไม่หวังดีได้ดาวน์โหลดไฟล์เว็บพนันมาไว้บนเซิร์ฟเวอร์ ไฟล์นั้นมีชื่อว่าอะไร?
**Answer:** `gambling.php`
**Investigation:**
พบคำสั่ง `wget -O ./gambling.php ...` ในประวัติการส่งคำสั่งผ่านช่องโหว่เว็บ

### ID 112: Hosting Gambling Website - 09
**Question:** ใช้คำสั่งใดใน Linux เพื่อแก้ไขไฟล์หน้าเว็บหลักและทำการ Redirect?
**Answer:** `sed`
**Investigation:**
พบการใช้คำสั่ง `sed -i` เพื่อแทรกแท็ก `<meta http-equiv="refresh" ...>` เข้าไปในไฟล์ `index.php` เพื่อให้ Redirect ไปยังหน้าเว็บพนัน

### ID 114: Hosting Gambling Website - 10
**Question:** ก่อนที่ผู้ไม่หวังดีจะใช้คำสั่งบางอย่างเพื่อแก้ไขไฟล์เว็บไซต์ พวกเขาได้ทำการใช้คำสั่งอะไรเพื่อสำรวจเนื้อหาของเว็บ ให้ตอบในรูปแบบถอดรหัส
**Answer:** `;ls -al;`
**Investigation:**
จากการตรวจสอบ Access Log ของเครื่อง Web Server พบว่าผู้ไม่หวังดีส่งคำสั่งผ่านพารามิเตอร์ `secret` ในรูปแบบ URL Encoded (`%3Bls+-al%3B`) ซึ่งเมื่อถอดรหัสออกมาจะได้คำสั่ง `;ls -al;` ที่ใช้ในการแสดงรายการไฟล์และสิทธิ์ในไดเรกทอรี

### ID 117: Hosting Gambling Website - 11
**Question:** ไฟล์หนึ่งที่ถูกแก้ไขเป็นอันดับแรก จงระบุ ชื่อไฟล์ ดังกล่าว ซึ่งเป็นจุดเริ่มต้นของการโจมตี ตัวอย่างเช่น (robots.txt)
**Answer:** `index.php`
**Investigation:**
จากการตรวจสอบคำสั่งที่ผู้ไม่หวังดีใช้ผ่านช่องโหว่ Command Injection พบการใช้คำสั่ง `sed` เพื่อแก้ไขไฟล์ `index.php` เป็นอันดับแรก เพื่อทำการ Redirect ผู้ใช้งานไปยังหน้าเว็บพนัน

### ID 119: Hosting Gambling Website - 12
**Question:** นอกจากการแก้ไขไฟล์เดิมแล้ว ผู้ไม่หวังดีได้ทำการเพิ่มไฟล์ใหม่ที่ไม่เคยมีมาก่อนเข้ามาในระบบด้วย จากการตรวจสอบ FIM Alert ทั้งหมด พบว่ามีไฟล์ที่ถูก "เพิ่มใหม่" เข้ามาบนเว็บเซิร์ฟเวอร์เป็นจำนวนทั้งหมดกี่ไฟล์ ให้ตอบเป็น (number)
**Answer:** `2`
**Investigation:**
จากการตรวจสอบระบบตรวจจับการเปลี่ยนแปลงไฟล์ (FIM) พบว่ามีการแจ้งเตือนการเพิ่มไฟล์ใหม่ (Added) จำนวน 2 ครั้ง ซึ่งตรงกับคำสั่ง `wget` ที่ดาวน์โหลดไฟล์ `gambling.php` และ `gam.jpg` มาติดตั้งบนเซิร์ฟเวอร์
